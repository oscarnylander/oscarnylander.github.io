Languages I enjoy

I've worked with a large number of languages during my career, but I'm currently favoring the following languages:

  • Rust - I hope one day that Rust becomes viable for all use cases. The language without compromises.
  • Swift - While the language is slightly skewed towards Apple platforms, it's still quite well-designed and pleasant to work with.
  • TypeScript - Should I ever need to write something to run on JavaScript runtimes, this is my weapon of choice.
